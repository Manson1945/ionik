Integración API.

Escrito por ionikSend, el día 12 de Octubre 2024.

Actualizado el 12 de Noviembre 2024

1. Con el Administrador de cuenta ingrese a la sección "API" que se encuentra en el panel principal.

2. En esta sección encontrará las credenciales para conectarse de forma segura con la plataforma.

API Key: Identifica la integración que realiza las solicitudes. Es obligatoria para todas las operaciones que usan la API de IonikSend.

API Token: Clave para autenticar el Acceso a la API.

Route: Es el canal asignado para el envío de mensajes.

Chatbot: Indica el identificador del Bot que gestionará las conversaciones.
